Interactive hospital operations dashboard built with Python, Dash, and Plotly for exploring hospital service performance across departments, weeks, staffing levels, patient satisfaction, and acceptance rates.

## Overview

This project focuses on visualizing hospital operations data using coordinated interactive views. The dashboard allows users to compare departments, inspect weekly trends, explore multivariate relationships, and analyze how staffing configurations may affect service performance.

## Features

- Interactive dashboard built with Dash and Plotly
- Coordinated multiple views
- Weekly filtering and zooming
- Department comparison
- Parallel coordinates visualization
- Staff configuration / what-if analysis
- Responsive layout with custom CSS styling

## Technologies

- Python
- Dash
- Plotly
- Pandas
- NumPy
- CSS
